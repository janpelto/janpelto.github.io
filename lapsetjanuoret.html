Linkkejä oppaisiin ja peleihin

Code Combat

Code Combat on selaimessa toimiva opetuspeli, jossa lapsi oppii koodaamaan HTML, Python ja Javascript kielillä, minkä lisäksi tutuksi tulevat tietojenkäsittelytieteiden perusteet. Peliin voi rekisteröityä, mutta se ei ole välttämätöntä, sillä peliä voi pelata varsin pitkälle ilman kirjautumistakin.

Koodikirja

Koodikirja on Juha Paanasen kirjoittama opas, jossa käydään läpi ohjelmoinnin perusperiaatteita, minkä lisäksi oppaassa on myös pieniä tehtäviä, joilla vahvistetaan osaamista.

Code.org

Code.org on sivusto, jossa opetellaan koodaamisen perusteita pelimäisellä tyylillä, kun käyttäjä pääsee esimerkiksi liikuttelemaan hahmoja. Sivustolla ei kirjoiteta koodia ollenkaan, vaan ohjelmointi tapahtuu siirreltävien palikoiden avulla, joita voi yhdistellä toisiinsa.
