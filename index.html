C#

C# on Microsoftin kehittämä olio-ohjelmointikieli, joka julkaistiin
kesäkuussa 2000. Kieli kehitettiin yhdistämään
C++:n tehokkuus ja Visual Basicin tuottavuus.

Python

Python on jo 1980 luvun loppupuolella kehitetty ohjelmointikieli. Se on yksi yleisimmistä
kielistä, mitä aloittelijat opettelevat, sen helpon syntaksin ja vähäisten opeteltavien yksityiskohtien takia.

HTML

HTMl ei ole varsinaisesti ohjelmointikieli, vaan kuvauskieli ja sitä käytetään webbi-sivujen toteuttamisessa.
Ensimmäiset kuvaukset HTML:sta julkaistiin vuoden 1991 loppupuolella.

Java

Java on vuonna 1995 julkaistu, yksi maailman suosituimmista olio-ohjelmointikielistä maailmassa.
Sitä käytetään mm. web- ja android kehityksessä, servereissä, työpöytäsovelluksissa
sekä pelikehityksessä.

Javascript

JavaScript vuonna 1995 kehitetty skriptikieli, jonka tarkoitus on lisätä webbi-sivuille toiminnallisuuksia, sekä vuorovaikutusta käyttäjän kanssa.

Scratch

Scratch on vuonna 2003 kehitetty ohjelmointikieli, joka on suunnattu ensimmäiseksi
ohjelmointikieleksi erityisesti lapsille ja nuorille. Scratchin avulla lapset
pääsevät tutustumaan ohjelmoinnin maailmaan.
